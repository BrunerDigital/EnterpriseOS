# Customization Guide

## Brand

Update `brand` in `lib/marketing.ts`, the logo mark in `components/marketing/marketing-shell.tsx`, and CSS theme variables in `app/globals.css`.

## Pricing

Edit `pricingPlans` in `lib/marketing.ts`.

## Copy

Edit page entries in `marketingPages`, `landingPages`, `funnelTemplates`, `flowSteps`, and `copyLibrary`.

## Visual Style

The current system uses a dark premium SaaS aesthetic with cyan, emerald, and restrained violet accents. Adjust gradients, borders, and panels in `app/globals.css` and `components/marketing`.
