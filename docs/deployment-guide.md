# Deployment Guide

1. Run `npm install`.
2. Run `npm run typecheck`.
3. Run `npm run build`.
4. Replace all `brunerdigital.example` URLs.
5. Add environment variables for database, auth, analytics, email, SMS, payments, and CRM webhooks.
6. Deploy to Vercel or a compatible Next.js platform.
7. Verify sitemap, robots, metadata, forms, analytics events, and mobile layouts.
